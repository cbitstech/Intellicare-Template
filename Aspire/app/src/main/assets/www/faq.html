1. What is a "value?"

    A value is a way to live that we identify as core to our identity and happiness. By practicing behavior that accords with our values, we can have happier and fuller lives.

  2. Can I have only one path per value?

    You may have multiple paths. Select "Edit Paths" in the main screen to add additional paths to the current value.

  3. How do I add paths to a value?

    After selecting "Edit Paths," select "Add Path" from the upper right corner of the screen.

  4. Can I change the picture for a value?

    Yes. Tap the text in the lower right corner of the image that says "Tap here to select a new image." You can then choose a picture from your phone gallery. Try adding images that have a personal meaning to you for that value!

  5. Will the paths stay here forever? Is there a way to delete or remove them? Can I edit them?

    To edit a path, select "edit paths" from the main screen. A tap on a path in the list will allow you to edit the wording. A long press will allow you to delete the path.

  6. Why are there faces along the bottom of the home screen? Some of them are smiling and some are not. What do they mean?

    The faces indicate if you have checked off any paths that day. Tapping on a smiling face will allow you to see all of the paths that you completed on that day/ If you had not checked off any paths for any values, you will see "No paths recorded this day..."

  7. Does a neutral face mean I should set up new paths or can I do ones that I have already set up?

    To cause the faces to smile, check off paths that you have followed for each virtue using the "check in" button.

  8. How do I send suggestions or questions to the developers?

    This can be done right from the app. Just click the menu button in the top right corner of the home screen and choose "Feedback." Now choose how you would like to send your comments, type away, and send it off.
