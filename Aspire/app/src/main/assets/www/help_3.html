How to use Aspire

In Aspire, you will first be given a list of possible values that people like you have given us and asked which ones apply to you. You can also add your own value if it's not included on the list. Then you will specify some "paths" or activities that you can engage in that help you live that value. Again, we'll give you some suggestions that people have found useful, but please create your own, as these will hold more meaning. The paths we've suggested are somewhat general, but when you create your own paths it helps them be as personal for you possible.

As you complete these paths each day you can log that in the Aspire app and it will keep track of your progress overtime. If you completed a path on each day a smiley face will indicate that on the home screen. You can always try new values and paths.
