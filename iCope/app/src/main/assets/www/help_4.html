How often to use iCope

You can use iCope in a variety of ways. The first time you use it, you can think about the next few days and write Coping Reminders for any stressful or difficult situations you might encounter (for example, a difficult conversation, something you need to get done but often avoid). Each day, it's useful to take some time and plan for the upcoming day.
